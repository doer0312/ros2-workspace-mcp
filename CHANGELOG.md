# Changelog

All notable changes to this project are documented here in a concise
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/) style.

## [0.1.0] - 2026-07-11

### Added

- Seven read-only MCP analysis tools for ROS 2 workspaces.
- One fixed workspace-summary Resource and one package-context Resource Template.
- The `review_ros2_workspace` guided review Prompt.
- Canonical root sandboxing, symbolic-link containment, and bounded UTF-8 reads.
- Static Python AST, XML, safe-YAML, CMake, interface, URDF, and unexpanded Xacro analysis.
- Cross-platform unit, regression, and real stdio integration tests.
